# 🧠 Use Cases: Copilot for Neurodiversity
[↑ Back to README](../README.md)

</br>

This page provides practical Copilot use cases and ready-to-use prompts to support neurodivergent individuals (e.g. ADHD, autism, dyslexia).

Each example includes:
- the challenge
- why Copilot helps
- a copy-paste-ready prompt

</br>

## ⚡ ADHD (Focus, Prioritization, Task Initiation)

### 🧩 Challenge
- Difficulty starting tasks and organizing work
- Overwhelm when facing unstructured information
- Struggles with prioritization and attention

### ✅ Why Copilot helps
- Breaks down complexity into small, actionable steps
- Creates structure and prioritization automatically
- Reduces cognitive load and decision fatigue

</br>

### 💬 Prompt: Task Breakdown
```
Break this task into a clear, step-by-step plan.
Limit to 5–7 steps.
Make each step actionable and easy to start.
Highlight the very first action I should take.
```
</br>

### 💬 Prompt: Daily Focus Plan
```
Create a simple focus plan for today based on these tasks: [paste tasks].

- Group tasks into 2–3 priority blocks
- Suggest realistic time slots
- Keep it simple and achievable
```
</br>

### 💬 Prompt: Anti-Overwhelm Summary
```
Summarize this content into a short, structured list:
- Key points (max 5)
- Next actions (max 3)
Keep it clear and easy to scan.
```
</br>

## 🧩 Autism (Clarity, Predictability, Communication)

### 🧩 Challenge
- Difficulty interpreting tone and implicit meaning
- Uncertainty in vague or ambiguous instructions
- Need for clear structure and predictability

### ✅ Why Copilot helps
- Translates ambiguity into explicit instructions
- Clarifies tone and intent
- Provides structured, predictable outputs

</br>

### 💬 Prompt: Clarify Communication
```
Rewrite this message in a clear and explicit way:
- Remove ambiguity
- Use direct and precise language
- Highlight what is expected from me
```
</br>

### 💬 Prompt: Tone & Intent Explanation
```
Explain the tone and intent of this message.

- What does the sender likely expect?
- Is there anything implicit I should be aware of?
- Suggest a clear and appropriate response
```
</br>

### 💬 Prompt: Step-by-Step Instructions
```
Turn this into a clear instruction guide:
- Step-by-step format
- No assumptions
- Include expected outcome for each step
```
</br>

## 📖 Dyslexia (Reading, Writing, Comprehension)

### 🧩 Challenge
- Difficulty reading long or complex texts
- High effort for writing structured and correct content
- Struggles with clarity and flow

### ✅ Why Copilot helps
- Simplifies language and reduces complexity
- Generates structured drafts
- Improves clarity, grammar, and readability

</br>

### 💬 Prompt: Simplify Text
```
Rewrite this text in plain language:
- Use short sentences
- Use simple words
- Structure it clearly
Keep the meaning but make it easier to read.
```
</br>

### 💬 Prompt: Structured Summary
```
Summarize this text into:
- 3–5 bullet points
- simple wording
- clear structure
```
</br>

### 💬 Prompt: Writing Support
```
Create a clear and well-structured draft based on these notes: [paste notes].

- Use simple language
- Keep sentences concise
- Ensure correct grammar and flow
```
</br>

## 🔄 Cross-Use Prompts (Universal Support)

</br>

### 💬 Prompt: Turn Chaos into Structure
```
Organize this information into:
- clear sections
- bullet points
- logical order

Make it easy to understand at a glance.
```
</br>

### 💬 Prompt: Guided Thinking

```
Help me think through this step by step.

- Ask me 3–5 guiding questions
- Then suggest a clear next step
```
</br>

### 💬 Prompt: Reduce Cognitive Load

</br>

Simplify this as much as possible:
- remove unnecessary detail
- highlight only essentials
- present in an easy-to-scan format

</br>

## ✅ Key Principles

- Prefer structure over free text
- Limit output to small, manageable chunks
- Use clear, explicit language
- Focus on next actions, not just information
- Reduce cognitive load wherever possible
