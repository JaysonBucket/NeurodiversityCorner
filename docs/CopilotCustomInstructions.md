## You define how AI should behave.
Custom Instructions let you set the rules once instead of correcting the output every time.
This is useful when you already know what kind of answers work best for you.  

Copilot can become significantly more helpful when it adapts to your individual preferences, thinking style, and recurring needs. 
[This section helps you extract those patterns from your past conversations](LearnFromConversations.md) and turn them into reusable customizations.

<br>

## When Custom Instructions are the right choice

Use Custom Instructions when:

- you want a consistent default structure  
- you want predictable responses  
- you want less variation  
- you know what format or tone works best for you  

This is especially useful when you repeatedly ask for the same guardrails.

<br>

## Step by step

The basic Custom Instructions flow in Microsoft 365 Copilot is:

1. Open Copilot Chat and sign in with your work or school account.  
2. Select **... Copilot chats and more** in the top-right corner.  
3. Open **Settings**.  
4. Go to **Personalization**.  
5. Find **Custom instructions**.  
6. Select **Edit instructions**.  
7. Add the instructions you want Copilot to follow by default.  
8. Save the instructions.  

<br>


## Examples

| Problem | Why Custom Instructions are the right choice | What to set |
|---|---|---|
| Answers are too long and exhausting to process | This is about how AI should respond by default, not what it should remember about you | `Always keep answers concise. Start with the key point. Avoid long paragraphs unless I explicitly ask for detail.` |
| You need structure to stay focused | This defines a reusable response format that should apply consistently | `Always use bullet points and clear headings when possible.` |
| Too many options increase overwhelm | This is a behavioural rule for how AI should present information | `Give me no more than 3 next steps at a time. Prioritise the simplest useful action first.` |
| The language feels too complex or too polished | This is about tone and wording, so it fits Custom Instructions better than Memory | `Use simple, direct language. Avoid overly polished or inflated wording.` |
| You often reshape outputs into action-oriented formats | This is a stable formatting preference and should not need to be repeated every time | `When possible, summarise first, then provide action-oriented details in a practical format.` |
| You lose focus when answers mix summary and detail too early | This is about the order in which AI should respond | `Always start with a short summary. Only then provide supporting detail.` |

<br>

## A strong baseline example

If you want one practical starting point, this is a good baseline:

```
Keep answers short, structured and practical.
Use bullet points instead of long paragraphs.
Use simple, direct language.
Give me a maximum of 3 next steps at a time.
Start with the most relevant point first.
```

<br>

## Guiding idea

Custom Instructions are about this:

> Do not fix every answer manually. Fix how answers are created.
