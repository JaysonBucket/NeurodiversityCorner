## AI can adapt to how you work over time.

Memory allows AI to learn patterns about you, so you do not have to repeat them every time.
This is useful when something describes how you generally work and keeps showing up again and again.

<br>

## When this helps

Some things are not one-time instructions.

They are patterns.

- you prefer short answers  
- you need structure to get started  
- you work best with checklists  
- you get overwhelmed by too much detail  
- you need clear definitions of “done”  

Instead of repeating this in every prompt, you can store it once.

<br>

## When Memory is the right choice

Use Memory when:

- the behaviour repeats  
- it reflects a personal preference  
- it describes how you generally work  
- you want support without having to ask every time  

Do not use Memory for one-time tasks.

❌ “Summarise this document”  
❌ “Write an email to X”

✅ “I prefer short, structured answers”  
✅ “I need tasks broken down into small steps”

<br>

## Typical situations

You might notice this:

- You keep asking to “make it shorter”  
- You always restructure answers yourself  
- You repeat the same instructions again and again  
- You wish AI would just “get how you work”  

This is where Memory helps.

<br>

## Step by step

The basic Memory flow in Microsoft 365 Copilot is:

1. Open Copilot Chat and sign in with your work or school account.  
2. In the compose box, tell Copilot to remember something about how you work.  
3. When saved, Copilot shows a **Memory updated** notice.  
4. If you want to manage saved memories later, open **... Copilot chats and more** → **Settings** → **Personalization** → **Manage saved memories**.  

<br>

## Examples

### 1) Problem: You lose energy filtering long answers
**Why Memory is the right choice**  
This is not a one-time issue. It is a recurring preference in how you consume information.

**What to save**
> Remember that I prefer short, structured answers with the key point first.

<br>

### 2) Problem: You get overwhelmed when tasks feel too large
**Why Memory is the right choice**  
This describes a recurring need in how you approach work, not just one specific task.

**What to save**
> Remember that I work better when tasks are broken into small, actionable steps.

<br>

### 3) Problem: You work best with checklists
**Why Memory is the right choice**  
This is a stable working preference you probably want across many different situations.

**What to save**
> Remember that checklist-style outputs are easiest for me to work with.

<br>

### 4) Problem: Starting is easier than finishing
**Why Memory is the right choice**  
This is a recurring pattern in how support should be shaped for you.

**What to save**
> Remember that I usually need more help finishing tasks than starting them.

<br>

## Guiding idea

Memory is about this:

> You do not adapt to AI every time. AI starts adapting to you.
